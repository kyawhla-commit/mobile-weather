# Install NetInfo Package

## Required Package

The offline mode feature requires `@react-native-community/netinfo` package.

## Installation

Run this command in your project directory:

```bash
npm install @react-native-community/netinfo
```

Or with yarn:

```bash
yarn add @react-native-community/netinfo
```

## For Expo Projects

If you're using Expo, run:

```bash
npx expo install @react-native-community/netinfo
```

## After Installation

1. Restart your development server
2. For iOS: `cd ios && pod install && cd ..`
3. Rebuild your app

## Verification

The app will automatically detect network changes and enable offline mode features once the package is installed.
